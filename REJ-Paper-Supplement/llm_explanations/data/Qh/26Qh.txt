**Constraints**:  
- The formula imposes a constraint on the next state when t is true in it (requiring y to be true there).  
- When t is false in the next state, no constraint is applied.  
- The condition must hold at every state in the execution.  
**Evaluation**:  
- At each state, check the next state. If t is true there, y must be true in that same next state.  
- If t is never true in any next state, the formula is trivially satisfied.
